# 📈 한국 주식 위젯

실시간 한국 주식 현재가를 화면에 항상 띄워두는 미니 위젯 프로그램입니다.

---

## ⚙️ 설치 방법

### 1. 라이브러리 설치
```bash
pip install PyQt6 requests
```
또는
```bash
pip install -r requirements.txt
```

### 2. 실행
```bash
python stock_widget.py
```

---

## 🖱️ 사용법

### 위젯
| 동작 | 설명 |
|------|------|
| 드래그 | 위젯을 원하는 위치로 이동 |
| **▼ 버튼** 클릭 | 평단가·수량·손익 정보 확장 (5초 후 자동 축소) |
| 우클릭 | 수정 / 삭제 메뉴 |

### 시스템 트레이 (작업표시줄 우측)
| 동작 | 설명 |
|------|------|
| 우클릭 → **종목 추가** | 새 종목 위젯 추가 |
| 우클릭 → **종료** | 프로그램 종료 |

---

## 📁 파일 구조
```
stock_widget.py   ← 메인 프로그램
requirements.txt  ← 필요 라이브러리 목록
stocks.json       ← 종목 설정 자동 저장 (실행 후 생성)
```

---

## ⚠️ 주의사항
- **시세 갱신 주기**: 30초마다 자동 갱신 (네이버 금융 API 사용)
- **장 마감 후**: 당일 종가가 표시됩니다
- **인터넷 연결** 필요

---

## 🖥️ .exe 파일로 변환하기 (배포용)

Python 없이 실행할 수 있는 단독 실행 파일을 만들려면:

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name 주식위젯 stock_widget.py
```

생성된 `dist/주식위젯.exe` 파일을 사용하면 됩니다.
